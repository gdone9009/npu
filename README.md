# 🧠 Mini NPU Simulator
> **AI 전용 프로세서(NPU)의 핵심 연산 메커니즘을 파이썬으로 구현하고 성능을 분석하는 시뮬레이터 프로젝트입니다.**

## 1. 프로젝트 개요 (Overview)
본 프로젝트는 인간의 시각적 패턴 인식 과정을 컴퓨터가 수치적으로 처리하는 **MAC(Multiply-Accumulate) 연산**의 원리를 이해하기 위해 기획되었습니다. 3x3부터 25x25까지 다양한 크기의 필터를 통해 입력 데이터의 유사도를 판별하고, 하드웨어적 연산 최적화의 필요성을 데이터로 증명합니다.

### 🎯 주요 학습 목표
* **MAC 연산 구현**: 2차원 배열 기반의 곱셈 및 누적 합산 로직 이해
* **부동소수점 정밀도 제어**: 허용오차(Epsilon) 기반의 수치 비교 정책 수립
* **데이터 정규화**: 다양한 라벨링 규칙을 표준 포맷으로 통합하는 전처리 로직 설계
* **성능 분석**: 데이터 크기 증가에 따른 시간 복잡도 $O(N^2)$ 분석

---

## 2. 기술 스택 및 실행 환경
* **언어**: Python 3.8+
* **라이브러리**: 표준 라이브러리(`json`, `time`)만 사용 (**외부 라이브러리 사용 금지**)
* **데이터 포맷**: JSON (패턴 및 필터 데이터 영속성 관리)

## 3. 핵심 기능 (Features)
1. **모드 1: 사용자 입력 (3x3)**
   - 사용자로부터 직접 필터와 패턴을 입력받아 즉시 판정 및 연산 시간 측정
2. **모드 2: data.json 일괄 분석**
   - 대용량 데이터 세트(5x5, 13x13, 25x25) 로드 및 일괄 테스트
   - 정규화 로직을 통한 `expected` 값과 판정 결과 비교 (PASS/FAIL)
3. **성능 리포트**
   - 크기별 평균 연산 시간 측정 및 이론적 연산 횟수 비교 표 출력

---

## 4. 구현 및 설계 방침
* **수치 비교 정책**: 부동소수점 오차를 고려하여 `abs(score_a - score_b) < 1e-9`인 경우 `UNDECIDED`로 판정합니다.
* **라벨 정규화**: `+` 또는 `cross`는 `Cross`로, `x` 또는 `X`는 `X`로 내부 표준화하여 판정의 정확도를 높입니다.
* **방어적 로직**: 사용자 입력 오류 시 재입력을 유도하며, JSON 스키마 불일치 시 해당 케이스만 FAIL 처리하고 프로그램은 중단 없이 실행됩니다.
